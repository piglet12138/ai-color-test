# AI Color Test · AI 私人形象顾问

一个探索性的个人项目：上传一张照片，AI 帮你**测色（12 季型）→ 配穿搭 → 形象测试（花纹/发色/发型/妆容/首饰）→ 在你本人身上生成预览**，并把结果存进你的历史档案。

## 特性
- **测色**：GPT-5 视觉按四季十二型体系判季型 + 个人色板 + 妆容建议。经 FairFace 评测纠偏（三轴独立判冷暖/明度/纯度，缓解“默认判秋/偏暖”偏差）。
- **穿搭造型**：成套配色 + 真人上身预览 + 小红书式点评。
- **形象测试**：花纹/发色/发型（男女）+ 妆容/首饰（分性别方案）。
- **氛围感妆容**：在本人脸上生成自然通透的氛围人像。
- **拍照采集**：应用内相机 + 拍照指引 + 客户端质量门（曝光/清晰度/削波/分辨率）+ A4 白纸白平衡；低置信时冷暖输出区间并引导重拍。
- **用户系统 + 历史**：注册/登录，结果按类型汇总、可看详情。

## 技术
- 后端：零依赖 Node（`server.mjs`）+ JSON 存储；统一走一个 OpenAI 兼容反代（GPT-5 多模态 + gpt-image 编辑）。
- 前端：原生 JS，移动端 App 外壳（`public/app.html` + `app-ui.js`）。
- 参考数据：`presets.mjs`（12 季型色板 + 选项目录）、`presets-makeup.mjs`（氛围妆风格）。

## 运行
```bash
cp .env.example .env   # 填反代 API_KEY
node server.mjs        # http://localhost:5178
```
